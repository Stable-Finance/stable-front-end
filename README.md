## Stable Monorepo

This contract holds the frontend for Stable Protocol. It is hosted at [https://app.trystable.com](app.trystable.com). Deployments are automatic using Cloudflare Pages and are triggered automatically on pushing to main on GitHub. This site is fully static and theoretically could be hosted on IPFS.

Intstall Dependencies: `pnpm install`
Run locally: `pnpm dev`
Build: `pnpm build`. Files in the directory `out` can be hosted statically.
